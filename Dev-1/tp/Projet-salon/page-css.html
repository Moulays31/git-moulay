Introduction du CSS :

CSS est l'acronyme de Cascading Style Sheet, ou feuille de style en cascade en français.
Le CSS permet d'insérer des styles sur un code HTML ou XHTMl et donc permet de définir très précisément le comportement de chaque élément de la page.
On voit déjà des sites fait entièrement en CSS (contrairement à la majorité des sites qui sont fait en tableau) mais le problème de la compatibilité avec certains explorateurs ne permettent pas d'exploiter toutes les fonctions des feuilles de style.

Exemple de css :

Modifier la couleur de la police :

partie HTML:

<p>class="rouge">text en rouge</p>

partie CSS:

.rouge{ color: rgb(255, 7, 7);}

texte en rouge

Modifier a police du texte:

partie HTML:

<p>class="italic">texte en italic et arial </p>

partie CSS:

.italic{

font-size: 150%;

font-style: italic;

font-family: Arial, Helvetica, sans-serif;

text-decoration:underline;

}

texte italic, grand, style arial et souligné.

les alignements:

On peut utiliser la fonction "float" dans le css pour pouvoir placer le texte apres une image.

il peut être utilisé de plusieurs manière.

left : Un mot-clé indiquant que l'élément doit flotter du côté gauche du bloc qui le contient.

right : Un mot-clé indiquant que l'élément doit flotter du côté droit du bloc qui le contient.

inline-start: Un mot-clé indiquant que l'élément doit flotter du côté du début du bloc qui le contient. Pour les scripts LTR (les langues qui s'écrivent de la gauche vers la droite), c'est le bord gauche ; pour les scripts RTL, c'est le bord droit.

inline-end: Un mot-clé indiquant que l'élément doit flotter du côté de la fin du bloc qui le contient. Pour les scripts LTR (les langues qui s'écrivent de la gauche vers la droite), c'est le bord droit ; pour les scripts RTL, c'est le bord gauche.

Box model :

Des balises de type block ont des dimensions précises

On utilise les propriétés (exprimées en % ou en px):

  • width (largeur du block)
  • height (hauteur du block)

Tous les blocs possèdent des Marges, on distingue 2 types :

Block (Marges ⬆ ➡ ⬇ ⬅ )

  • Marges Intérieures (propriété padding)
  • Marges Extérieures (propriété margin)

(les balise inline ont aussi des marges)

partie HTML:

<p>class="box">texte avec un box model bleu, une police de couleur blanche.
Un contour noir avec une marge de 60 pixels et le texte centré. </p>

partie CSS:

.box{

color: rgb(255, 255, 255);

width: 50%;

height: 50%;

background-color: blue;

}

texte avec un box model bleu, une police de couleur blanche. Un contour noir avec une marge de 60 pixels et le texte centré.

Dégradés :

Pour gérer les dégradés on va utiliser la propriété background-image

il existe 2 types de dégradés :

  • linear-gradient (Dégradés Linéaire)
    • Direction
    • MultiColor
    • transparence
    • espace entre les couleurs

  • radial-gradient (Dégradés Radial)
    • Positionnement
    • Rayonnement
    • Répétition

partie HTML:

<p>class="lineaire">texte avec un dégradé linéaire directionnelle MultiColor.Le pourcentage de vert est de 50% et il part d'en haut à droite. </p>

<p>class="radial">texte avec un dégradé radial rayonnement partant d'en haut </p>

partie CSS:

.lineaire{
background-image: linear-gradient(to right top,blue,green 50%,yellow);
width: 225px;
height: 100px;
position: relative;
top: 5px;
left: 5px;
}

.radial{
float:right;
background-image: radial-gradient(farthest-side at top,yellow,green,blue);
width: 225px;
position: relative;
height: 100px; position:relative;
}

texte avec un dégradé linéaire directionnelle MultiColor.Le pourcentage de vert est de 50% et il part d'en haut à droite.

texte avec un dégradé radial rayonnement partant d'en haut
